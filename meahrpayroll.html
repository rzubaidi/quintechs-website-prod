MEA HR and Payroll

Comprehensive Human Resources and Payroll Solution for Microsoft Dynamics 365

4. Deployment

The product is built using Dynamics 365 Finance and Operations. A model will be provided and can be deployed through LCS.

  • Current Application Release: Microsoft Dynamics 365 for Finance and Operations (10.0.35)
  • Platform Release: Update59 (7.0.6972.121)
  • Integration: Integration with external systems is supported using Rest APIs. Integration with Logic App / Power Automate is supported using business events.

5. Data Management

Our Payroll supports Data Management features to allow the upload of all employee information and other modules. We enhanced some functions to upload daily employee transactions, such as:

  • Overtime mass creation
  • Earning & Deduction mass creation
  • Loan Mass Creation
  • Absence mass creation

6. Support and Maintenance

Software Maintenance is categorized into four key areas:

  • Perfective: Monitoring and Bug Fixes.
  • Corrective: To fix errors.
  • Preventive: Includes Support.
  • Adaptive: Includes Enhancements.
